# Блог-приложение на Spring Boot

Это backend-приложение для ведения блога, реализованное на **Java** с использованием **Spring Boot**. Приложение
упаковывается в исполняемый JAR, предоставляет API для управления постами, комментариями,
изображениями.

---

## Команды

Сборка проекта:

`mvn clean package`

Запуск проекта:
`java -jar target/blog-app.jar`

Запуск тестов:

`mvn test`


Приложение будет доступно по корневому пути / (http://localhost:8080/)

---

## Стек технологий

- **Java 21**
- **Spring Web MVC** — для обработки HTTP-запросов
- **Spring JDBC** — для работы с базой данных
- **MapStruct** — для маппинга
- **Lombok** — для уменьшения boilerplate-кода
- **H2 Database** — встроенная СУБД
- **Maven** — система сборки
- **Spring Boot Actuator** — для мониторинга и информации о приложении
