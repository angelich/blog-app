# Блог-приложение на Spring Framework

Это backend-приложение для ведения блога, реализованное на **Java** с использованием **Spring Framework**. Приложение
развертывается в сервлет-контейнере **Apache Tomcat** и предоставляет API для управления постами, комментариями,
изображениями.

---

## Команды

Сборка проекта:

`gradlew clean war`

Запуск тестов:

`gradlew test`

Деплой в сервлет-контейнер (Tomcat):

`cp build/libs/ROOT.war /путь/к/tomcat/webapps/`

Приложение будет доступно по корневому пути / (http://localhost:8080/)

---

## Стек технологий

- **Java 21**
- **Spring Web MVC** — для обработки HTTP-запросов
- **Spring JDBC** — для работы с базой данных
- **MapStruct** — для маппинга
- **Lombok** — для уменьшения boilerplate-кода
- **H2 Database** — встроенная СУБД
- **Gradle** — система сборки c использованием Kotlin
